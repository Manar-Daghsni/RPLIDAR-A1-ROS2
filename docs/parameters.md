# Node Parameters

## lidar_node parameters
- `frame_id`: The TF frame for laser data (default: "laser")
- `angle_compensate`: Whether to compensate for angle (default: true)

## slam_node parameters
- `resolution`: Map resolution in meters/pixel (default: 0.05)
- `map_update_interval`: How often to update the map (default: 0.5s)
